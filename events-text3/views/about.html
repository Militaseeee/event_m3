About My Event Platform

My platform about events is a digital space where you can explore and manage a collection of events in a simple and user-friendly way. Whether you're a reader, student, or educator, this platform provides the tools to browse, register, and maintain your favorite events.

This app was developed using HTML, CSS, JavaScript (Vanilla), and JSON Server to simulate backend functionality. It demonstrates role-based access, CRUD operations, and a clean SPA experience.
